How to apply CSS to HTML

There are Three methods of applying CSS to an HTML file

External Stylesheet
Using an external CSS file with a .css extension is a popular way to style web pages. By linking this file to multiple pages, you can apply the same styling across all of them, making it easy to manage and update your website's appearance. Example 

              /* styles.css */
              body {
                  font-family: Arial, sans-serif;
                  background-color: #f0f0f0;
              }
              
              h2 {
                  color: blue;
              }
              

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>External CSS Example</title>
                <link rel="stylesheet" href="styles.css">
            </head>
            <body>
                <h2>Example Heading</h2>
                <p>This is a paragraph.</p>
            </body>
            </html>
Internal Stylesheet
We can also use an internal stylesheet which is made within the stm document, lik the name implaies and intenal stylesheet you place CSS inside a <style> element in the html <head> 
            
              <!DOCTYPE html>
              <html lang="en-GB">
                <head>
                  <meta charset="utf-8" />
                  <title>My CSS experiment</title>
                  <style>
                    h1 {
                      color: blue;
                      background-color: yellow;
                      border: 1px solid black;
                    }
      
                    p {
                      color: red;
                    }
                  </style>
                </head>
                <body>
                  <h1>Hello World!</h1>
                  <p>This is my first CSS example</p>
                </body>
              </html>
Inline Styles
Inline styles in html affects a single element in HTML, this shoudl be contained wintin a style element. An Example might be:\ 
              
                <!doctype html>
                <html lang="en-GB">
                  <head>
                    <meta charset="utf-8" />
                    <title>My CSS experiment</title>
                  </head>
                  <body>
                    <h1 style="color: blue; background-color: yellow; border: 1px solid black;">
                      Hello World!
                    </h1>
                    <p style="color: red;">This is my first CSS example</p>
                  </body>
                </html>

Java Script in a Webpage

JavaScript is a programminf language used to make webpages interactive, Its mostly to make a website visually appealing. There are two ways to call JavaScript code within HTML by using a script element.

Linking an external script
usually scriprs are used in their own .js files. If you need to execute a .js script from a webpage, we can use <script> with a src attribute which points to the script file. For example: 
<script src="path/to/my/script.js"></script>
Writing javascript within HTML
it is similar to linking a script but instead of creating a src file you can just add JavaScript code witing a <script> attribute. For example: 

            <script>
            window.addEventListener("load", () => {
              console.log("This function is executed once the page is fully loaded");
            });
          </script>
